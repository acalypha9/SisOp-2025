<h1>Sistem bilangan</h1>
<h2>Nomor 1</h2>
<p>Bilangan biner adalah bilangan yang berbasis <b>dua</b></p>
<h2>Nomor 2</h2>
<p>Konversikan bilangan desimal 1234<sub>10</sub> dalam bilangan biner</p>
<code>
1234 / 2 = 617 sisa 0
617 / 2 = 308 sisa 1
308 / 2 = 154 sisa 0
154 / 2 = 77 sisa 0
77 / 2 = 38 sisa 1
38 / 2 = 19 sisa 0
19 / 2 = 9 sisa 1
9 / 2 = 4 sisa 1
4 / 2 = 2 sisa 0
2 / 2 = 1 sisa 0
1 / 2 = 0 sisa 1</code>

Jadi, jawabannya adalah <b> 10011010010<sub>2</sub></b>
<h2>Nomor 3</h2>
<p>Konversikan bilangan biner 10101010<sub>2</sub> ke dalam bilangan desimal</p>
<code>1 x 2<sup>7</sup> + 0 x 2<sup>6</sup> + 1 x <sup>5</sup> + 0 x <sup>4</sup> + 1 x 2<sup>3</sup> + 0 x 2<sup>2</sup> + 1 x 2<sup>1</sup> + 0 x 2<sup>0</sup> = 170<sub>10</sub></code>
<h2>Nomor 4</h2>
<p>Konversikan bilangan biner 101011111001<sub>2</sub> ke dalam bilangan oktal</p>
<code>1012 = 5<sub>8</sub>
011<sub>2</sub> = 3<sub>8</sub>
111<sub>2</sub> = 7<sub>8</sub>
001<sub>2</sub> = 1<sub>8</sub>
Jadi, jawabannya adalah <b>5371<sub>8</sub></b></code>
<h2>Nomor 5</h2>
<p>Konversikan bilangan oktal 2170<sub>8</sub> ke dalam bilangan biner</p>
<code>2<sub>8</sub> = 010<sub>2</sub>
1<sub>8</sub> = 001<sub>2</sub>

0<sub>8</sub> = 000<sub>2</sub>

<h2>Nomor 6</h2>
<p>Konversikan bilangan desimal 1780<sub>10</sub> ke dalam bilangan heksadesimal</p>
<code>1780 / 16 = 111 sisa 4 -> 4<sub>16</sub>
111 / 16 = 6 sisa 15 -> F<sub>16</sub>
6 / 16 = 0 sisa 6 -> 6<sub>16</sub>
Jadi, jawabannya adalah <b>06F4<sub>16</sub></b></code>
<h2>Nomor 7</h2>
<p>Konversikan bilangan heksadesimal ABCD<sub>16</sub> ke dalam bilangan desimal</p>
<code>A->10 x (16)<sup>3</sup> = 40960
B->11 x (16)<sup>2</sup> = 2816
C->12 x (16)<sup>1</sup> = 192
D-> 13 x (16)<sup>0</sup> = 13
Jadi, jawabannya adalah 40960 + 2816 + 192 + 13 = <b>43981<sub>10</sub></b></code>
<h2>Nomor 8</h2>
<p>Konversikan bilangan pecahan desimal 0,3125<sub>10</sub> ke dalam bilangan biner</p>
<code>0,3125 x 2 = 0 sisa 0,625
0,625 x 2 = 1 sisa 0,25
0,25 x 2 = 0 sisa 0,5
0,5 x 2 = 1 sisa 0
Jadi, jawabannya adalah <b>0,0101<sub>2</sub></b></code>
<h2>Nomor 9</h2>
<p>Konversikan bilangan desimal 11,625<sub>10</sub> ke dalam bilangan biner</p>
<code>1. bilangan bulat <b>(11)</b>
11 / 2 = 5 sisa 1
5 / 2 = 2 sisa 1
2 / 2 = 1 sisa 0
1 / 2 = 0 sisa 1
1101<sub>2</sub>
2. bilangan pecahan <b>(0,625)</b>
0,625 x 2 = 1 sisa 0,25
0,25 x 2 = 0 sisa 0,5
0,5 x 2 = 1 sisa 0
0,101<sub>2</sub>
Jadi, jawabannya adalah <b>1101,101<sub>2</sub></b></code>
<h2>Nomor 10</h2>
<p>Konversikan bilangan desimal 1784,240<sub>10</sub> ke dalam bilangan heksadesimal</p>
<code>1. bilangan bulat <b>(1784)</b>
1784 / 16 = 111 sisa 8->8
111 / 16 = 6 sisa 15->F
6 / 16 = 0 sisa 6->6
6F8<sub>16</sub>
2. bilangan pecahan <b>(0,240)</b>
0,240 x 16 = 3->3 sisa 0,84

Jadi,jawabannya adalah <b>6F8,3D7<sub>16</sub></b></code>
<h2>Nomor 11</h2>
<p>Konversikan bilangan 010100011,001111101<sub>2</sub> ke dalam bilangan desimal</p>
<code>1. bilangan bulat biner <b>(010100011)</b>
0 x 2<sup>8</sup> + 1 x 2<sup>7</sup> + 0 x 2<sup>6</sup> + 1 x 2<sup>5</sup> + 0 x 4<sup>4</sup> + 0 x 2<sup>3</sup> + 0 x 2<sup>2</sup> + 1 x 2<sup>1</sup> + 1 x 2<sup>0</sup> = 163
163<sub>10</sub>
2. bilangan pecahan biner <b>(001111101)</b>
0 × 2<sup>-1</sup> + 0 × 2<sup>-2</sup> + 1 × 2<sup>-3</sup> + 1 × 2<sup>-4</sup> + 1 × 2<sup>-5</sup> + 1 × 2<sup>-6</sup> + 1 × 2<sup>-7</sup> + 0 × 2<sup>-8</sup> = 0.246
0,246<sub>10</sub>
Jadi,jawabannya adalah 163,125<sub>10</sub></code>
<h2>Nomor 12</h2>

<h2>Nomor 16</h2>
<p>Rubahlah Bentuk BCH F0DE<sub>16</sub> ke dalam bilangan biner</p>
<code>F<sub>16</sub> = 1514<sub>10</sub> -> 11111<sub>2</sub>

<h2>Nomor 17</h2>
<p>Nyatakan positip atau negatip bilangan biner 01111011<sub>2</sub></p>
<code>0 = Positif (+)
0111 1011= 123<sub>10</sub>
Jadi, jawabannya adalah Positip <b>123<sub>10</sub></b></code>
<h2>Nomor 18</h2>
<p>Nyatakan bilangan biner negatip 11110111<sub>2</sub> ke dalam bilangan desimal</p>
<code>1111 0111<sub>2</sub> (invert) -> 0000 1000<sub>2</sub>
0000 1000 + 1 = 0000 1001<sub>2</sub>
0000 1001<sub>2</sub> = 9<sub>10</sub>
Jadi, jawabannya adalah <b>-9</b></code>
Jadi, jawabannya adalah Positip <b>123<sub>10</sub></b></code>
<h2>Nomor 19</h2>
<p>Nyatakan ASCII Code 5A<sub>16</sub> dalam bentuk karakter</p>
<code>5 x 16<sup>1</sup> + 10 x 16<sup>0</sup> = 90<sub>10</sub> -> ‘Z’
Jadi, jawabannya adalah <b>Z</b></code>
<h2>Nomor 20</h2>
<p>Nyatakan Karakter ASCII 'm' dalam hexadesimal</p>
<code>‘m’ -> 109<sub>10</sub>
109 / 16 = 6 sisa 13 -> D<sub>16</sub>
6 / 16 = 0 sisa 6 -> 6<sub>16</sub>
Jadi, jawabannya adalah <b>6D<sub>16</sub></b></code>
